CSS3
Cascading Style Sheets

css can be applied to our html webpages in  3 ways:
	1)internal css==>saved as html file only
	2)external css==> saved as .css file
	3)inline css ==> style only for one line code

syntax of css code:
	
	selector{
		property:value;
		}

	ex:
	h1{
		color:red;
	}
		

properties list:
================
color:red;
font-family:arial;
font-size:16px;
text-align:left/center/right/justify;

border:1px solid red;
width:300px;
height:300px;

margin:30px; ==> margin is a space out side of the element
margin:0px 30px;

text-decoration:none;
background-color:yellow;

Padding:30px; ==>padding is a space inside of the element
padding:0px 30px;

RESPONSIVE WEB DESIGNING (RWD):

Responsive design is a graphic user interface (GUI) design approach used to create content that adjusts smoothly to various screen sizes. Designers size elements in relative units (%) and apply media queries, so their designs can automatically adapt to the browser space to ensure content consistency across devices

media queries:
=============

}
